# Casse-Briques

# Français
Projet de fin d'année de Terminale (2013).

## Lancer le jeu
### Windows
Lancer cassebrique.exe dans `Ressources`.

### Linux
Lancer cassebrique.exe dans `Ressources` avec Wine.

`wine Ressources/cassebrique.exe`
